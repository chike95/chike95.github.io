# 1.1 Ubuntu Server

  1. 选择系统语言-English

  1. 选择操作-Install Ubuntu Server

  1. 选择安装过程和系统的默认语言-English

4.选择区域-other

  1. 选择亚洲-Asia

  1. 选择国家-China

  1. 选择字符集编码-United States

  1. 是否检测键盘布局，选择No

  1. 选择键盘类型-English (US)

10.选择键盘布局-English (US)

  1. 配置网络，选择Continue

  1. 选择手动配置网络-configure network manually

  1. 设置主机ip: 192.168.30.100

  1. 设置主机子网掩码: 255.255.255.0

  1. 设置主机网关: 192.168.30.1

  1. 设置主机dns: 192.168.30.1

  1. 设置主机名(项目名称-taizhou)

  1. 设置域名-可以选择不填，直接Continue

  1. 设置用户名（tonglei）-Continue

  1. 设置登陆账号（tonglei）-Continue

  1. 设置登录密码（空格选择“Show Password in Clear”可以显示密码）-Continue

  1. 重复上一步设置的登录密码-Continue

  2. 是否使用弱密码，选择是-Yes

  1. 是否加密home文件夹，选择否-No

  1. 确认时区是否正确(这里是“Asia/Shanghai”或者“Asia/Shanghai”都可以)，选择是-Yes

  1. 选择分区方式(分区向导-使用整个磁盘和设置逻辑分区LVM)-“Guided-use entire disks and set up LVM”

也可以不分区：“Guided-use entire disk”

  1. 选择要分区的磁盘(这里只有一块)-“SCSI3 （0,0,0）(sda) -53.7GB VMware Virtual disk”

  1. 是否将变更写入磁盘，选择是-Yes

  1. 用于指导分区的卷组数量,默认即可，选择-Continue

  1. 是否写入变更的磁盘，选择是-Yes

  1. 设置HTTP代理，无需填写直接下一步-Continue

  1. 设置系统升级方式，选择没有自动更新-No automatic updates

  1. 选择要安装的软件，选择：lamp server 和 OpenSSH Server，然后下一步-Continue

  1. 是否安装GRUB引导程序，选择是-Yes

35.完成安装，选择下一步-Continue

36.系统安装完会自动启动主机

37.输入设置好的登录账户和密码就可以开始使用了

硬盘 /dev/sdb1 上包含的是 NTFS 文件系统。如果你想将其格式化为 ext4 文件系统，需要先确认是否要继续进行该操作。

请注意，格式化会删除硬盘上的所有数据，因此在执行格式化之前，请确保你已经备份了重要的数据。如果你确定要继续格式化操作，请输入 y 并按下 Enter 键。如果你不想继续，请输入 N 并按下 Enter 键。

（2）创建一个用于挂载的目录。运行以下命令创建 /srv/data 目录：

sudo mkdir /srv/data
1

（3）将硬盘挂载到 /srv/data 目录。运行以下命令：

sudo mount /dev/sdb1 /srv/data
1

这将临时挂载硬盘到 /srv/data 目录。

（5）如果你希望系统重启后仍然保持挂载状态，需要更新 /etc/fstab 文件。运行以下命令打开 /etc/fstab 文件进行编辑：

sudo nano /etc/fstab
1

（6）在文件的末尾添加以下行：

/dev/sdb1  /srv/data  ext4  defaults  0  2
1

保存并关闭文件（使用 Ctrl + X，然后按 Y 和 Enter）。

（7）如果您想要卸载挂载的硬盘，可以执行以下命令：

sudo umount /srv/data
1

（8）要查看硬盘的使用状态，你可以使用 df 命令。运行以下命令可以获取硬盘的使用情况：

df -h
1

这将显示系统中所有挂载点的磁盘空间使用情况，包括硬盘的使用情况。

如果你只想查看特定硬盘（例如 /dev/sdb1）的使用情况，可以运行以下命令：

df -h /dev/sdb1
1

这将仅显示 /dev/sdb1 硬盘的使用情况。请注意，/dev/sdb1 在查看之前必须已经被挂载。

# 1.3 网络管理

# 1.3.1 修改 IP

ls /etc/netplan
1

01-netcfg.yaml [文件名具体看情况]

sudo vim 01-netcfg.yaml 按照如下内容编辑（冒号后面有空格）

(enp5s0:注意需要通过ifconfig –a 确认)

# This file describes the network interfaces available on your system
# For more information, see netplan(5).
network:
  version: 2
  renderer: networkd
  ethernets:
    enp1s0:
      dhcp4: no
      addresses: [192.168.30.100/24]
      optional: true
      gateway4: 192.168.30.1
      nameservers:
              addresses: [192.168.30.1]
    enp2s0:
      dhcp4: true
